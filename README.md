# gerenciamento-de-pedidos
Sistema em Python para gerenciamento de pedidos personalizados de Bubble Tea. Permite escolha de bases (leite, maracujá, rosa, manga) e complementos (boba, lichia, etc.), com descontos por tipo de cliente e cashback de 10%. Usa POO, JSON para persistência e modularização. Projeto acadêmico UEFS.
